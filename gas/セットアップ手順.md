# GAS セットアップ手順

## 1. スプレッドシートにGASを追加

1. スプレッドシートを開く
2. メニュー →「拡張機能」→「Apps Script」
3. `コード.gs` の内容をまるごとコピペして保存（Ctrl+S）

## 2. 定数を設定

`コード.gs` の先頭2行を編集：

```javascript
const SS_ID      = '1t4g1uyTRXr9acoNK_bE-uLGtzq9ZazHVCghn2Lpx-SM'  // ← 今回のスプレッドシートID
const LINE_TOKEN = 'ここにLINE Notifyトークンを貼る'
```

## 3. LINE Notify トークンの取得

1. https://notify-bot.line.me/ja/ にアクセス
2. 右上「マイページ」→「トークンを発行する」
3. 通知先：塾長のLINEアカウント or 管理用グループ
4. 発行されたトークンを `LINE_TOKEN` に貼る

## 4. シートを自動生成

1. GASエディタで関数を `setupSheets` に切り替えて実行
2. スプレッドシートに以下が追加される：
   - 📋 講師マスタ
   - ⏱ 打刻ログ
   - 📝 勤務記録ログ

## 5. 講師マスタにデータを入力

| LINE_ID | スタッフID | 氏名 | グレード | メール |
|---|---|---|---|---|
| U〇〇〇〇〇 | S001 | 田中 花子 | B3 | hanako@... |

※ LINE IDは講師のLINEアカウントから確認（開発者が確認するか、講師に聞く）

## 6. Web App としてデプロイ

1. GASエディタ右上「デプロイ」→「新しいデプロイ」
2. 種類：「ウェブアプリ」
3. 設定：
   - 説明：塾講師管理API
   - 実行ユーザー：自分
   - アクセスできるユーザー：**全員**（LINEアプリからアクセスするため）
4. 「デプロイ」→ URLをコピー

## 7. LINEアプリに設定

`.env` ファイルを編集：

```
VITE_DEV_MODE=false
VITE_LIFF_ID=あなたのLIFF ID
VITE_GAS_URL=https://script.google.com/macros/s/〇〇〇/exec
```

## 8. 深夜バッチのトリガー設定

1. GASエディタ左メニュー「トリガー（時計アイコン）」
2. 「トリガーを追加」
3. 設定：
   - 関数：`nightlyBatch`
   - イベントのソース：時間主導型
   - 時間の種類：日タイマー
   - 時刻：午前2時〜3時
4. 保存

これで毎晩2時に各先生シートが自動更新されます。
